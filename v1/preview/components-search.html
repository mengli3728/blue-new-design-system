Usage Guidelines

Panel search: Use .search-bar inside sidebars or filter panels. Includes a search icon SVG and an input with placeholder.

Global search: Scale .search-bar to 320px+ for top-bar search. Use max-width to control responsive width.

Focus state: The .search-bar:focus-within pseudo-class switches border to var(--color-primary) when the input is focused.
