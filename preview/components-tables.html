Usage Guidelines

Layout: Wrap any table in .table-wrap. The wrapper ships with built-in 20px padding and overflow scroll — drop it directly inside a card or panel without adding another padding layer. Table width is 100% inside the wrapper.

Headers: Use <thead> with <th> — headers get page background, medium weight, and auto word-wrap prevention.

Row hover: table tr:hover td gets var(--color-primary-bg) highlight. Add cursor: pointer on rows for clickable tables.

Status cells: Use .badge-* inside table cells for status indicators with colored dots.

Don't double-pad: Avoid wrapping .table-wrap in a second padded container — the 20px is part of the system contract, stacking padding breaks alignment with sibling cards.
