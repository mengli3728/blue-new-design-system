Usage Guidelines

Gradient bar (.nav-topbar): Use as the primary global navigation on branded pages. Place logo-white.png on the left, followed by top-level navigation items. The active page item receives a full-height solid #2C7CF5 background. Add right-side action icons (.nav-actions) for notifications, settings, and user — icons are 16px line-art white SVGs with 12px gap and 20px right margin.

Light bar (.nav-topbar-light): Use for interior pages, settings panels, or pages where the gradient bar is visually heavy. Place logo-default.png on the left. Text and icons use var(--text-heading) (#000000), active text uses #2C7CF5. Active and hover states both use #ECF7FF background.

Sidebar (.nav-sidebar + .nav-item): Use for secondary or multi-level navigation. Active items use primary color background with white text. Items collapse to icon-only on viewports narrower than 1024px. All sidebar nav items use 8px border-radius for hover and selected states.

Sidebar — Blue Theme (.sidebar-blue-item): Use for branded side navigation with full primary-color background. All navigation items display white text (#FFFFFF) on var(--color-primary) (#2C7CF5) background. Hover state transitions to rgba(255,255,255,0.10). Active items use #0A64EA background — no border. Item padding is 10px 20px with 2px 8px margin from the container edge — 20px distance from sidebar edges. Ideal for settings panels, profile pages, and branded side menus.
