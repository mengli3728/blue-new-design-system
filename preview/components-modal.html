Usage Guidelines

Overlay: Use .modal-overlay for full-screen backdrop with var(--bg-overlay) (rgba(0,0,0,0.5)).

Modal sizing: Default width 480px, max-width 90vw, max-height 80vh. Aspect ratio recommended 4:3 or 2:1.

Structure: .modal-header (title + close, 12px padding top/bottom) → full-width divider (var(--border-color)) → .modal-body (scrollable content) → .modal-footer (actions, right-aligned).

Toast: Auto-dismiss notifications after 3 seconds.
