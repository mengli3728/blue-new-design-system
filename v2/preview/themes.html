主题色 / Themes

博瑞蓝 设计体系内置 3 套主题色：博瑞蓝、国网绿、南瑞蓝。 它们共享语义色、文字色、间距、字体、阴影与动效 token，仅 primary 色族（含顶部导航渐变与 sidebar 选中底色）切换。 使用方式：在 <html data-theme="…"> 上锁定单一主题；或在顶部导航放置 .theme-switcher 下拉选择器让用户运行时切换。

Token 对照

data-theme="blue"

使用方式 / How to use
单主题 — 创建时锁定

<html data-theme="nari">      <!-- 南瑞蓝 -->
